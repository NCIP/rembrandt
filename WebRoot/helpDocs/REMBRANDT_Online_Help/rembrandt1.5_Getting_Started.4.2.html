Creating a User Account
Each REMBRANDT user is given a unique user name and password. The user name and password you are assigned determines your access rights for the software. To set up a user account, you must:
•
Contact NCICB Application Support:
–
NCICB@pop.nci.nih.gov
–
888-478-4423 (toll-free) or 301-451-4384 (local)
OR
•
Go to the NCICB REMBRANDT login page and click the request username/password link to send an e-mail requesting a username and password to NCICB Application Support.
To log in to REMBRANDT, see Logging In.
