Results Overview
The View Results page shows a collection of reports previously viewed in a particular user session. This allows you to compare reports by opening them in separate windows. For example, you can compare clinical and gene expression reports with a set of patient samples. You can view Clinical reports (see Clinical Reports), results generated with the Advanced Search function (see Advanced Search or Query Results) and the High Order Analysis function (see High Order Analysis Results).
The View Results page also enables you to download BRB Array tools and the static, archive files for use in BRB-ArrayTools (see Downloading BRB Array Tools and Files).
